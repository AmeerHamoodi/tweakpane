Tweakpane

Monitor

Tweakpane has a feature for monitoring a property change. Use addMonitor() or addColorMonitor() (for color value) to monitor it.

var pane = new Tweakpane();

pane.addMonitor(target, 'time');        // String
pane.addMonitor(target, 'active');      // Boolean
pane.addMonitor(target, 'wave');        // Number

pane.addColorMonitor(target, 'color');  // Color

Graph

addGraph() option provides a graph component to watch a number property.

var pane = new Tweakpane();
pane.addGraph(target, 'wave', {
  min: -5.0,
  max: +5.0,
  count: 200
});

min is a minimum value of the graph, max is a maximum value, and count is data count to plot.

Logger

To log a single-line string property, use addLogger.

var pane = new Tweakpane();
pane.addLogger(target, 'time', {
  count: 10
});

count option is a count of data to hold.

Multi-line text

To monitoring a multi-line string property, enable multiline option.

var pane = new Tweakpane();
pane.addMonitor(target, 'message', {
  multiline: true
});

Changing monitoring interval

You can change an interval for monitoring by interval option, in the order of milliseconds. The default value is 100ms.

var pane = new Tweakpane();
pane.addMonitor(target, 'wave', {
  interval: 500,
  label: 'wave (0.5s)'
});
pane.addMonitor(target, 'wave', {
  interval: 1000,
  label: 'wave (1.0s)'
});
