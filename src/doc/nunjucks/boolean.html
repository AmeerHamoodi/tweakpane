Tweakpane provides a checkbox field for a boolean property by default.

var params = {
  debug: true
};
var pane = new Tweakpane();
pane.addCheckbox(params, 'debug');

Selector

You can use a named list for boolean value. Pass an array of two string values (true, false) to values option to addSelector().

var pane = new Tweakpane();
pane.addSelector(target, 'debug', {
  values: ['Enabled', 'Disabled']
});
