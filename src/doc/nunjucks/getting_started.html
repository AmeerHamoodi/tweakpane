1. Add Tweakpane to your project

First, download the latest Tweakpane and import it to your web page.

<!-- In the head tag -->
<link rel="tweakpane-{{ config.version }}.min.css">

...

<!-- In the body tag -->
<script src="tweakpane-{{ config.version }}.min.js"></script>

2. Create a pane

Create an instance of Tweakpane by new Tweakpane().

<script>
var pane = new Tweakpane();
</script>

3. Ready to go!

Add components to fine-tune a property, monitoring a property, etc. See the reference on the left side for more detailed information.

// ...

var pane = new Tweakpane();
pane.addControl(target, 'color');
pane.addControl(target, 'speed');

// ...
