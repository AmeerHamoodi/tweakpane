Tweakpane provides a text field for a string property.

var params = {
  name: 'cocopon',
};
var pane = new Tweakpane();
pane.add(params, 'name');

List

If you want to choose a value from list, pass a string list as list option:

var pane = new Tweakpane();
pane.add(target, 'direction').list([
  'left',
  'up',
  'right'
  'down'
]);

An object that has string values is also supported:

var pane = new Tweakpane();
pane.add(target, 'vimKey').list([
  left:  'H',
  up:    'K',
  right: 'L',
  down:  'J'
]);
